Torch7

Scientific computing for LuaJIT.

About Torch7

Torch7 is a scientific computing framework with wide support for machine learning. It is easy to use and provides a very efficient implementation, thanks to an easy and fast scripting language, LuaJIT, and an underlying C implementation.

Among other things, it provides:
  • a powerful N-dimensional array, torch.Tensor
  • lots of routines for indexing, slicing, transposing, ...
  • amazing interface to C, via LuaJIT
  • linear algebra routines

If you have questions or need support, check out the mailing list.

Installation

The easiest way to install Torch is to follow these instructions. This simple script will install LuaJIT, Luarocks and build Torch as well as a few important packages for you.

Torch, and all its related packages, are now distributed as simple Luarocks packages. It only requires LuaJIT (>= 2.0) and Luarocks (>= 2.0.12):

$ luajit --version
Luarocks 2.0.12
$ luajit -v
LuaJIT 2.0.2 -- Copyright (C) 2005-2013 Mike Pall. http://luajit.org/
$ luarocks --server=https://raw.github.com/torch/rocks/master install torch
...

Other packages can be installed the same way, using Luarocks:

$ luarocks --server=https://raw.github.com/torch/rocks/master install image
$ luarocks --server=https://raw.github.com/torch/rocks/master list

Packages

Torch used to be distributed as a single repo, which provided a few default packages. These packages, along with their documentation, are now each hosted on their own Github repository. Here is the list of official Torch packages:

  • Torch7 - doc
  • nn - doc
  • paths - doc
  • ...

More packages can be found on Torch's Github portal.

Even more packages can be found using Luarocks:

$ luarocks --server=https://raw.github.com/torch/rocks/master search --all
